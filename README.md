# Lovaly Dream Resort

Welcome to the Lovaly Dream Resort project! This is a fully responsive landing page built with Next.js, Tailwind CSS, and Framer Motion, designed to showcase the luxurious offerings of our resort, hotel, and restaurant.

## Project Structure

The project is organized as follows:

```
lovaly-dream-resort
├── app
│   ├── layout.tsx          # Main layout of the application
│   └── page.tsx            # Entry point for the landing page
├── components
│   ├── AboutSection.tsx     # Animated description of the resort
│   ├── ContactForm.tsx      # Stylish contact form component
│   ├── Footer.tsx           # Footer with contact information
│   ├── GalleryGrid.tsx       # Masonry/grid layout for images
│   ├── HeroSection.tsx      # Full-screen hero section
│   ├── Navbar.tsx           # Sticky navigation bar
│   └── ServicesCards.tsx    # Floating cards for services offered
├── styles
│   └── globals.css          # Global styles and Tailwind CSS imports
├── package.json             # npm configuration file
├── postcss.config.js        # PostCSS configuration for Tailwind CSS
├── tailwind.config.ts       # Tailwind CSS configuration
├── tsconfig.json            # TypeScript configuration
├── next-env.d.ts            # Type definitions for Next.js
└── README.md                # Project documentation
```

## Getting Started

To get started with the Lovaly Dream Resort project, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd lovaly-dream-resort
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser and navigate to:**
   ```
   http://localhost:3000
   ```

## Features

- **Responsive Design:** The landing page is fully responsive and looks great on all devices.
- **Animations:** Smooth animations using Framer Motion for an engaging user experience.
- **Glassmorphism Effects:** Stylish glassmorphism effects for cards and sections.
- **Easy Navigation:** Sticky navigation bar for easy access to different sections of the page.

## Technologies Used

- **Next.js:** A React framework for building server-rendered applications.
- **Tailwind CSS:** A utility-first CSS framework for styling.
- **Framer Motion:** A library for animations in React applications.
- **Lucide React:** A collection of icons for use in the application.
